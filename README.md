# 🚀 Personal Portfolio Website

A modern, responsive **AI & Data Science student portfolio website** built using HTML and CSS.
This project showcases personal details, technical skills, projects, certifications, and contact information in a clean and structured format.

---

##  Project Overview

This portfolio is designed to represent a professional digital identity. It highlights:

*  Personal introduction and career focus
*  Educational background and achievements
*  Technical skills and tools
*  Projects with descriptions and technologies used
*  Certifications and volunteering experience
*  Contact section for professional outreach

The design follows a **modern dark theme with glassmorphism effects**, smooth layout structure, and responsive behavior for different screen sizes.

> The project is built using **pure HTML and CSS**, making it lightweight, fast, and easy to deploy.  

---

## 🛠️ Technologies Used

## ⚙️ Setup Instructions

Follow these steps to run the project locally:

1. Download or clone the repository:

   ```
   gh repo clone narasimhabusireddy018/PORTFOLIO-
   ```

2. Navigate to the project folder:

   ```
   cd your-repo-name
   ```

3. Make sure all files are in the same directory:

   * `index.html`
   * `style.css`

## ▶️ How to Run the Project

There is no backend or installation required.

Simply:

1. Open the project folder
2. Double-click on **index.html**

OR

Right-click → Open with browser (Chrome / Edge / Firefox)
